# Agraseni Hospital Website — Apple Theme
## File Structure

agraseni-hospital/
├── index.html      ← Main website (paste all HTML here)
├── style.css       ← All styling (Apple dark theme)
├── script.js       ← Animations & interactions
└── README.md       ← This file

## How to Use

1. Create a folder called `agraseni-hospital` anywhere on your computer.
2. Create 3 files inside it:
   - `index.html`  → paste the HTML code
   - `style.css`   → paste the CSS code
   - `script.js`   → paste the JavaScript code
3. Open `index.html` in any modern browser (Chrome, Edge, Safari, Firefox).

## Features Included

- Apple-inspired dark design with animated gradient orbs
- Sticky navigation that blurs on scroll (macOS-style)
- Animated hero section with hospital stat counters
- Facilities grid with hover effects
- 17 Department cards with gradient underline animation
- Doctor profiles with image fallback avatars
- Scrolling insurance ticker (infinite loop)
- Gallery with lightbox on click
- Google Maps embed (dark-filtered to match theme)
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll, reveal-on-scroll animations
- Emergency call button always visible

## No Dependencies Required

This is a pure HTML/CSS/JS website.
No npm, no build tools, no frameworks needed.
Just open index.html in a browser — done.

## To Deploy Online

Upload all 3 files to any web host:
- Netlify (drag & drop — free)
- GitHub Pages (free)
- Any shared hosting (cPanel file manager)
